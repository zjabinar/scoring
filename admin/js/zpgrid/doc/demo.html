The Zapatec AJAX Grid Demos
Stock Performance
A basic example showing a grid with performance data of the stock of various companies.
Phone Bill
This phone bill example shows a more advanced capabilities of the Zapatec AJAX Grid, including paging, and limiting the data displayed. Also, the Zapatec Slider is used to define range filters.
Load Grid from Url
Using the Zapatec AJAX Transport to fetch the grid data after the page is displayed. The grid data can be defined as HTML table, JSON, or XML, making it easy to generate using any server side technology.
Yahoo Search
Fetch the data from an external source, in this case a yahoo search, and pass it the data to the grid.
The data is displayed in search result layout demonstrating that the output does not need to be tabular.
Show Data AS IS
Show data in the grid AS IS. You can preserve your original data format and still use the powerful Zapatec grid features. This is very useful to migrate your old style tables to the feature rich Zapatec Grid.
This demo also features how to flatfile (export) the grid into CSV format (or any other format). This export allows the user to import ANY Zapatec Grid into their own program, i.e. Excel.
Splice
Grid splice method allows to Add, Modify, and Delete grid data. You can specify exactly where you want the grid data changed. A useful example is to use the Splice and Flatfile features to tune your grid data and export grid to be imported into Excel.
Query Updates
See how you can query the grid to Insert, Replace, and Delete grid cells.
Charts
Use Flash to have charts in your grid.
Scrolling Grid
You can have the Zapatec Grid scroll vertically and horizontally. This is similar to the Window Freeze Panes option in Excel.
